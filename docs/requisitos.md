# Ágili Service Desk

## Objetivo

Desenvolver um sistema web para gerenciamento de solicitações internas de uma empresa, permitindo o controle e acompanhamento de chamados técnicos.

---

# Requisitos Funcionais

RF01 - O sistema deve permitir cadastrar categorias.

RF02 - O sistema deve permitir listar categorias.

RF03 - O sistema deve permitir editar categorias.

RF04 - O sistema deve permitir excluir categorias.

RF05 - O sistema deve permitir cadastrar chamados.

RF06 - O sistema deve permitir listar chamados.

RF07 - O sistema deve permitir editar chamados.

RF08 - O sistema deve permitir excluir chamados.

RF09 - O sistema deve permitir alterar o status de um chamado.

RF10 - O sistema deve permitir definir a prioridade de um chamado.

RF11 - O sistema deve exibir um dashboard com indicadores.

RF12 - O sistema deve permitir filtrar chamados por status e prioridade.

---

# Requisitos Não Funcionais

RNF01 - O sistema deve ser desenvolvido utilizando ASP.NET Core MVC.

RNF02 - O banco de dados será SQL Server.

RNF03 - O acesso aos dados será realizado utilizando Entity Framework Core.

RNF04 - A interface deverá utilizar Bootstrap.

RNF05 - O código deverá seguir boas práticas de organização.
